# GPT Partei Website

Vollständige Website der GPT Partei (Gemeinschaft · Progress · Tatkraft)

## Enthaltene Dateien

### Hauptseiten
- **index.html** - Startseite (leitet automatisch weiter)
- **Parteipunkte_Detail.html** - Hauptprogramm mit allen 14 Punkten
- **Finanzierung.html** - Detaillierte Finanzierung in 2 Phasen
- **Parteimitglieder.html** - Teammitglieder und Funktionen

### Assets
- **GptLogo.jpg** - Parteilogo

### Backup-Versionen (optional)
- Finanzierung_alt.html
- Finanzierung_backup.html
- GPT_Partei_Website.html
- Parteibogen_Final.html

## Verwendung

### Lokal öffnen
Einfach `index.html` im Browser öffnen - alle Links funktionieren relativ.

### Als Website hosten
Gesamten `website` Ordner auf Webserver hochladen (z.B. GitHub Pages, Netlify, Vercel).

## Navigation
- Startseite → Parteipunkte_Detail.html (14 Programmpunkte)
- Oben rechts: Links zu Finanzierung und Team
- Alle Seiten haben Navigation zurück zum Hauptprogramm
